Cum publici modificările pe site:
  1. Apasă Descarcă produse.json sus.
  2. Pune fișierul descărcat peste D:\Proiecte\Elixir\js\produse.json.
  3. Dacă ai adăugat imagini noi, pune-le în D:\Proiecte\Elixir\img\produse\.
  4. În terminal: git add . && git commit -m "Actualizare parfumuri" && git push
  5. În ~1 minut, site-ul live se actualizează.
